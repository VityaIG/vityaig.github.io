Changelogs

A clearer timeline of how WebBreak.ru became a physics breaking sandbox.

Release 1.1.0 Current

WebBreak.ru got a cleaner identity, stronger physics, mobile fixes, a tougher captcha flow, and low-end performance work.

Renamed Minebuild to WebBreak.ru and refreshed the greeting, tips, and changelog copy around the new domain. Reworked spawned-object physics, Build Mode snapping, Edit Mode selection, throwing, rotation, and mobile controls. Redesigned captcha UI, restored multiple auth captcha variants, removed unused code, and added low-end PC optimizations.
Release 1.0.0 Previous

WebBreak.ru is now a full building playground with accounts, profile search, terminal tools, object editing, and stronger physics.

Added Start Building, draggable Terminal, Build Mode, Edit Mode, object selection, and quick edit buttons. Added Supabase profiles, Account IDs, search pages, likes, dislikes, descriptions, refresh, and admin tools. Rebuilt object physics on Matter.js, then kept mobile comfort, Midnight Theme, rotating tips, and register captcha.
